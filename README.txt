# Engineering Graphics and Design Lab - Exam Question Generator

This is a simple web-based tool for generating randomized Engineering Graphics and Design Lab exam questions. It is specifically developed for internal lab exams at Jamia Hamdard University.

## Features

- Collects basic student information.
- Displays exam instructions clearly.
- Generates one compulsory question (Q1) and two randomized questions from a predefined bank.
- Restricts students to a maximum of three attempts, showing only the latest questions after that.

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

1. Open `index.html` in a browser.
2. Enter the required student details.
3. Click "Generate Questions" to display a new set.
4. A maximum of three attempts are allowed.
5. Press "Restart" to reset the form.
